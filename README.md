# 📚 Book Swapping Platform (MERN Stack)

## 📌 Inspiration
We built this platform to solve a common problem we faced.  
After buying books (stories, novels, etc.), they often remain unused after reading.  
Buying books repeatedly is not always affordable, and excessive screen reading can cause stress.  

This platform encourages **book sharing**, making reading more accessible and reducing digital strain.

---

## 📌 Overview
A full-stack web application that allows users to exchange books with others.  
Users can list books, browse available books, and request swaps through an interactive system.

---

## 🚀 Key Features

### 🔐 User System
- User authentication (Signup/Login)

### 📚 Book Management
- Add and list books
- Browse available books

### 🔄 Book Swap System
- Request book swap
- Owner can **accept/reject requests**
- Swap details include **location & time**

### 📧 Email Integration
- When a user requests a book, an **email is sent to the owner**
- Contains swap details (location, time, etc.)

### ⏳ Due Date & Penalty System
- Return deadline for books
- Penalty points if not returned on time

### ⭐ Feedback System
- Users can give feedback on books

---
